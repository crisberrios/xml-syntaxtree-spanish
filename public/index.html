Uso

  • <Nodo>: El elemento Nodo acepta las siguientes propiedades:
    • id: utilizado como identificador global del nodo. Ej: nodo1, raíz, etc
    • texto: texto del nodo
    • rasgos: separados por coma, agrega rasgos al nodo
    • st: st="true" raya el texto (strikethrough)
    • techo: esta propiedad indica que es un nodo techo. Un nodo techo no debe contener nodos hijo ejemplo: <Nodo texto="nn" techo />

Modificadores de texto

  • \1 para barra. Ej: V\1 = V'
  • _ para subíndice. Ej: V_o
  • \textsubscript para subíndice sin itálicas. Ej: A\textsubscript{ABC} o A\textsubscript B
  • ^ para superíndice. Ej: V^x. Entre llaves, ej: v^{PX} para múltiples caracteres en superíndice
  • \textsuperscript para superíndice sin itálicas. Ej: A\textsuperscript{ABC} o A\textsuperscript B
